# Neck Firming Cream - One Product Store

## Overview
Build a modern, high-converting one-product store for a neck firming cream. This is a dropshipping product targeting women 35-65 who are concerned about sagging neck skin, "turkey neck," and aging d√©colletage.

## Competitor Reference
See `assets/competitor-reference.jpg` - This is PEAQ Beauty AH-8, a competitor product. We're creating our OWN brand, not copying theirs.

## Brand Details
- **Brand Name:** Eleve (elegant, French-inspired)
- **Product Name:** Eleve Neck Firming Cream
- **Tagline:** "Elevate Your Beauty" or similar French-inspired elegance
- **Price:** $89 (original $149) - 40% OFF limited time
- **Key Ingredients:** Peptides, Retinol, Hyaluronic Acid, Vitamin E

## Design Requirements

### Overall Vibe
- Modern, clean, luxurious feel
- Warm cream/gold color palette (similar to competitor)
- Mobile-first responsive design
- Trust-building elements throughout

### Sections Needed (in order)

1. **Hero Section**
   - Large product image (use a placeholder for now, elegant gold/cream tube)
   - "LIMITED TIME: 40% OFF" badge
   - Compelling headline
   - Short benefit-driven subheadline
   - CTA button: "Shop Now - 40% Off"

2. **Problem/Agitation Section**
   - Address the pain: sagging neck, wrinkles, "turkey neck"
   - Before/after style imagery area (placeholders)
   - Emotional copy that resonates with target audience

3. **Solution/Product Features**
   - Key ingredients with benefits
   - How it works (3-step or science-backed explanation)
   - Clinical results claims (e.g., "87% saw visible improvement in 4 weeks")

4. **Social Proof**
   - Customer testimonials (4-6)
   - Star ratings
   - Before/after photos area
   - "As Seen On" trust badges (placeholder)

5. **How To Use**
   - Simple 3-step application guide
   - Morning/evening routine

6. **FAQ Section**
   - Accordion style
   - Common questions about shipping, results timeline, ingredients, returns

7. **Final CTA Section**
   - Urgency: "Only X left in stock"
   - Money-back guarantee badge
   - Secure checkout badges
   - Big CTA button

8. **Footer**
   - Minimal: links to terms, privacy, contact
   - Trust badges

### Technical Requirements
- Single HTML file with embedded CSS (Tailwind via CDN is fine)
- No external dependencies beyond CDN
- Smooth scroll behavior
- Mobile responsive
- Fast loading

### Image Strategy
Use high-quality Unsplash images for:
- Hero product shot (or elegant placeholder)
- Women in target age range looking confident
- Skincare/luxury lifestyle imagery
- Before/after placeholder areas

## Conversion Elements
- Sticky "Buy Now" button on mobile
- Countdown timer (optional)
- Stock scarcity indicator
- Multiple CTAs throughout page
- Trust badges near checkout buttons

## DO NOT
- Use the competitor brand name (PEAQ)
- Make medical claims that could get the ad account banned
- Use before/after images that look too "edited"
- Make the design look cheap or scammy
