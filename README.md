# LiftLux Skincare - NeckRevive Pro Store

A high-converting one-product landing page for a neck firming cream, targeting women 35-65.

## Overview

This is a modern, mobile-first e-commerce landing page designed for maximum conversions. Built with Tailwind CSS via CDN for fast loading and easy customization.

## Features

- **Mobile-First Design**: Responsive layout that looks great on all devices
- **Sticky Mobile CTA**: Always-visible purchase button on mobile
- **Trust Elements**: Reviews, guarantees, and social proof throughout
- **FAQ Accordion**: Interactive accordion for common questions
- **Smooth Scrolling**: Elegant navigation between sections
- **Luxury Aesthetic**: Warm cream/gold color palette with elegant typography

## Sections

1. **Hero** - Product showcase with discount badge and main CTA
2. **Trust Bar** - Ratings, customer count, and key badges
3. **Problem/Agitation** - Addresses pain points with before/after imagery
4. **Solution/Features** - Ingredients and clinical results
5. **Social Proof** - Customer testimonials and media mentions
6. **How To Use** - 3-step application guide with pro tips
7. **FAQ** - Common questions in accordion format
8. **Final CTA** - Purchase section with pricing and guarantees
9. **Footer** - Links and payment badges

## Tech Stack

- HTML5
- Tailwind CSS (CDN)
- Google Fonts (Cormorant Garamond, Montserrat)
- Vanilla JavaScript (accordion, smooth scroll)

## Usage

Simply open `index.html` in a browser. No build step required.

## Customization

- Colors are defined in the Tailwind config at the top of the file
- Fonts can be changed via Google Fonts link
- Images use Unsplash URLs and can be replaced with actual product photos
- Copy and pricing can be edited directly in the HTML
